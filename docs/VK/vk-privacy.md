# VK — защита данных

## Настройка приватности профиля

### Кто видит мои записи на стене

1. Настройки → Приватность
2. Раздел **Основное**
3. Выберите **Кто видит мои записи**
4. Рекомендуем: **Только я** или **Друзья**

### Кто видит мои фотографии

1. Настройки → Приватность → **Профиль**
2. Выберите **Кто видит мои фотографии**
3. Рекомендуем: **Только я** или **Друзья**

### Кто видит список моих друзей

1. Настройки → Приватность → **Профиль**
2. Выберите **Кто видит список моих друзей**
3. Рекомендуем: **Только я**

### Кто может писать мне личные сообщения

1. Настройки → Приватность
2. Выберите **Связь со мной**
3. Рекомендуем: **Друзья и друзья друзей** или **Только друзья**

### Кто может комментировать мои записи

1. Настройки → Приватность
2. Выберите **Комментарии**
3. Рекомендуем: **Друзья** или **Только я**

## Кто может найти меня по номеру телефона

1. Настройки → Приватность
2. Выберите **Кто может найти меня по номеру телефона**
3. Рекомендуем: **Только друзья** или **Никто**

## Как скрыть профиль из поиска

1. Настройки → Приватность
2. Выберите **Кому виден мой профиль в поиске**
3. Рекомендуем: **Только пользователям VK**

## Как ограничить отправку приглашений в сообщества

1. Настройки → Приватность
2. Выберите **Приглашения в сообщества**
3. Рекомендуем: **Только друзья**

## Что делать с подозрительной активностью

Если вы заметили:
- Незнакомые комментарии или записи от вашего имени
- Отправленные сообщения, которые вы не писали

**Немедленно:**
1. Завершите все сессии (Настройки → Безопасность → Завершить все сессии)
2. Смените пароль
3. Проверьте список активных устройств
4. Сообщите в поддержку VK

## Готово

Теперь ваш профиль VK защищён от посторонних.
