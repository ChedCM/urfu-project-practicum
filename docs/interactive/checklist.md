# Чек-лист «Проверь свой аккаунт за 5 минут»

## Вопросы

Отметьте варианты, которые подходят вам:

### 1. Двухфакторная аутентификация
- [ ] Включена на всех аккаунтах (+3 балла)
- [ ] Включена частично (+1 балл)
- [ ] Не включена (0 баллов)

### 2. Уникальные пароли
- [ ] Для каждого сервиса свой пароль (+2 балла)
- [ ] Повторяю пароли на некоторых сайтах (+1 балл)
- [ ] Один пароль везде (0 баллов)

### 3. Проверка сессий
- [ ] Проверяю раз в месяц (+2 балла)
- [ ] Проверял когда-то (+1 балл)
- [ ] Никогда не проверял (0 баллов)

### 4. Приватность номера телефона
- [ ] Видят только я (+2 балла)
- [ ] Видят только контакты (+1 балл)
- [ ] Видят все (0 баллов)

### 5. Осторожность со ссылками
- [ ] Никогда не перехожу по подозрительным ссылкам (+1 балл)
- [ ] Перехожу, но не ввожу данные (0 баллов)
- [ ] Перехожу и ввожу данные (-1 балл)

### 6. Менеджер паролей
- [ ] Использую (+1 балл)
- [ ] Не использую (0 баллов)

## Подсчёт баллов

| Баллы | Уровень защиты |
|-------|---------------|
| 0–4 | 🔴 Низкий — срочно настройте защиту |
| 5–8 | 🟡 Средний — есть куда улучшать |
| 9–11 | 🟢 Хороший — продолжайте следить |

## Что делать дальше

- 🔴 **0–4 балла**: Начните с включения 2FA во всех аккаунтах
- 🟡 **5–8 баллов**: Настройте приватность и уникальные пароли
- 🟢 **9–11 баллов**: Проверяйте сессии раз в месяц
