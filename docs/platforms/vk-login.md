# VK — вход и базовая защита

## Включение двухфакторной аутентификации

1. Откройте **Настройки** → **Безопасность**
2. Нажмите **Подтверждение входа**
3. Выберите способ получения кода:
   - Приложение для двухфакторной аутентификации (рекомендуется)
   - СМС-код
4. Сохраните **резервные коды** в надёжном месте

## Проверка активных сессий

1. Настройки → Безопасность → **Показать историю активности**
2. Проверьте список устройств
3. Нажмите **Завершить все сессии**, если видите незнакомые устройства

## Смена пароля

1. Настройки → Безопасность → **Пароль**
2. Введите текущий пароль
3. Придумайте новый пароль:
   - Минимум 8 символов
   - Используйте буквы, цифры и спецсимволы
   - Не используйте пароль от других сервисов
