# MAX — приватность

## Кто может звонить вам

1. Профиль → Приватность → **Входящие звонки**
2. Выберите: **Только контакты** (рекомендуется)

## Кто может найти вас по номеру

1. Профиль → Приватность → **Найти меня по номеру**
2. Выберите: **Только контакты** или **Никто**

## Приглашения в группы

1. Профиль → Приватность → **Приглашения в группы**
2. Выберите: **Только контакты**

## Статус «в сети»

1. Профиль → Приватность → **Статус онлайн**
2. Выберите: **Только избранные** или **Никто**

## Фильтр контента 18+

1. Профиль → Приватность → **Контент**
2. Включите **Безопасный**

## Проверка активных устройств

1. Профиль → Приватность → Пароль и вход
2. Нажмите **Посмотреть активные устройства**
3. Завершите незнакомые сессии

> 🔁 Рекомендуется проверять раз в месяц.
