# Telegram — базовая защита

## Включение двухфакторной аутентификации (Облачный пароль)

1. Настройки → Конфиденциальность → **Облачный пароль**
2. Нажмите **Включить двухэтапную аутентификацию**
3. Придумайте пароль (минимум 6 символов)
4. Добавьте подсказку и email для восстановления
5. Сохраните **резервные коды**

## Проверка активных сессий

1. Настройки → Устройства
2. Проверьте список активных сессий
3. Завершите незнакомые сессии кнопкой **Завершить**

## Настройка кода блокировки

1. Настройки → Конфиденциальность → **Код-пароль**
2. Включите и задайте 4-значный код
3. Настройте автоблокировку (рекомендуется: 1 минута)
