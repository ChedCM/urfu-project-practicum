# VK — настройка приватности

## Кто видит мои записи

1. Настройки → Приватность → **Кто видит мои записи**
2. Рекомендуем: **Только я** или **Друзья**

## Кто видит мои фотографии

1. Настройки → Приватность → **Кто видит мои фотографии**
2. Рекомендуем: **Только я** или **Друзья**

## Кто может писать мне сообщения

1. Настройки → Приватность → **Связь со мной**
2. Рекомендуем: **Друзья и друзья друзей**

## Кто может найти меня по номеру телефона

1. Настройки → Приватность → **Кто может найти меня по номеру телефона**
2. Рекомендуем: **Только друзья** или **Никто**

## Что делать при подозрительной активности

Если заметили незнакомые действия:
1. Завершите все сессии
2. Смените пароль
3. Включите двухфакторную аутентификацию
4. Сообщите в поддержку VK
