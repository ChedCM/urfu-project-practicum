# Словарь терминов

## 2FA (двухфакторная аутентификация)
Метод защиты, при котором для входа требуется пароль + дополнительный код (из SMS, приложения или биометрии).

## Активная сессия
Подключение вашего аккаунта к устройству или браузеру. Через сессию можно пользоваться аккаунтом без повторного ввода пароля.

## Брутфорс
Метод взлома перебором всех возможных паролей. Защищайтесь длинными сложными паролями.

## Фишинг
Поддельная страница входа, созданная для кражи логина и пароля. Выглядит как настоящий сайт.

## Социальная инженерия
Метод обмана, при котором злоумышленник выманивает пароль у самого пользователя (звонок «из поддержки», фальшивые письма).

## Менеджер паролей
Программа, которая хранит пароли в зашифрованном виде, помогает генерировать надёжные пароли и автоматически подставлять их при входе.
