# Практические рекомендации по безопасности учётных записей

## О проекте

Этот сайт создан для обычных пользователей социальных сетей и мессенджеров. Мы не используем сложные технические термины. Только понятные пошаговые инструкции.

## Для кого этот сайт

- Вы пользуетесь Telegram, VK или MAX
- У вас нет IT-образования
- Вы хотите защитить свой аккаунт, но не знаете с чего начать
- Вы уже сталкивались с подозрительным входом или потерей доступа

## Что вы найдёте на сайте

| Раздел | Что внутри |
|--------|-------------|
| Telegram | Базовая защита, приватность, скрытые чаты |
| VK | Настройки входа, защита данных |
| MAX | Двухфакторная аутентификация, безопасный режим, приватность |
| Интерактивы | Чек-лист и тест для проверки вашего аккаунта |
| Справочник | Инструкция по использованию, словарь терминов, что делать при взломе |

## Сколько времени займёт

- Прочитать инструкцию для одной соцсети: 5–10 минут
- Выполнить все рекомендации: 15–30 минут

## Нужна ли регистрация

Нет. Сайт работает без регистрации, входа и передачи личных данных.

## Начните прямо сейчас

Выберите социальную сеть в меню слева и следуйте инструкциям.
