# Инструкция по использованию ресурса

## Для кого этот сайт

Сайт создан для обычных пользователей социальных сетей и мессенджеров, у которых нет профильного IT-образования.

## Как устроен сайт

Сайт разделён на несколько блоков:

### 1. Рекомендации по платформам

Отдельные разделы для **Telegram**, **VK** и **MAX**.

В каждом разделе — пошаговые инструкции:
- Как включить двухфакторную аутентификацию
- Как проверить активные сессии
- Как настроить приватность

### 2. Интерактивные проверки

- **Чек-лист** — ответьте на вопросы и узнайте, насколько защищён ваш аккаунт
- **Тест** — проверьте свои знания о безопасности

### 3. Справочные материалы

- **Словарь терминов** — объяснение сложных слов
- **Что делать при взломе** — алгоритм восстановления доступа

## Как читать инструкции

Каждая инструкция выглядит так:

1. **Заголовок** — что вы научитесь делать
2. **Зачем это нужно** — короткое объяснение
3. **Пошаговые действия** — пронумерованный список
4. **Важно** — предупреждения

## Сколько времени займёт

- Прочитать инструкцию для одной соцсети: 5–10 минут
- Выполнить все рекомендации: 15–30 минут

## Нужна ли регистрация

Нет. Вы можете пользоваться сайтом без регистрации, входа и передачи личных данных.
